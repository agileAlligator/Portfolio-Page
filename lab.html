The lab

The claim: a guardrail that lives in the model's judgment fails, and a guardrail that lives in deterministic server code holds. Each exhibit here tests that claim a different way. One maps the attack surface, one lets you walk defenses up a control ladder until they fall, and one is a real server on the public internet, waiting.

// result

Result, so far: prompt-level defenses in these exhibits fall to a good payload, every time. The deterministic wall holds: 0 leaks in 105 attempts at the L3 rung in the lab run, plus a live count of held calls on the public server. The pattern is small enough to memorize: canonicalize, confine, allowlist, in server code where the model gets no vote.
